# 伤官 · 破界创新

> 十神之一，主理破界、创新与非常规思维。
> 承担系统的因果推理与创新算法职责。

## 模块组成

| 文件 | 功能 |
|------|------|
| `innovator.py` | 破界创新器，支持组合/迁移/逆向/突破四种创新模式 |

## 快速开始

```python
from tengod.伤官_破界创新 import Innovator

inv = Innovator()

# 组合创新
inv.combine(["AI", "区块链", "IoT"], "三者融合的智能合约系统")

# 迁移创新
inv.transfer("游戏化", "教育", "将游戏机制迁移到在线教育平台")

# 逆向创新
inv.reverse("用户点击越多越好", "思考减少用户点击的方案")

# 查看最优创意
print(inv.top_ideas(3))
print(inv.report())
```

## 核心特性

- **四种创新模式**：组合、迁移、逆向、突破
- **可行性 × 影响力**：双维度评估每个创意
- **自动排序**：按综合得分输出最优创意
- **类型筛选**：支持按创新类型查询
