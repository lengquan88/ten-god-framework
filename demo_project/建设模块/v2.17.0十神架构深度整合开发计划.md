### 阶段 2：十神模块深度集成（P1，预计 5-7 天）

**目标**：将 12 个十神模块从独立骨架升级为协同工作的完整系统

#### 2.1 核心链路集成

```
用户请求 → 正官_法度调度(API网关) → 元辰_本源定位(路由定位)
         → 正财_知识固化(知识检索) → 偏财_奇招演化(智能搜索)
         → 食神_创生输出(内容生成) → 伤官_破界创新(创新优化)
         → 七杀_品质裁决(质量审核) → 太极_阴阳调和(结果平衡)
         → 正印_滋养守护(配置管理) → 劫财_攻防边界(安全审查)
         → 偏印_桥接通变(外部适配) → 比肩_架构协同(插件生态)
         → 响应
```

#### 2.2 各模块集成任务

| 模块 | 当前状态 | 目标状态 | 关键任务 |

#### 2.3 集成测试

为每个模块编写集成测试，验证模块间调用链路：

```python
# 示例：正官→元辰→正财→食神 链路测试
def test_full_pipeline_bazi_calculation():
    """端到端：用户请求 → 路由 → 知识检索 → 内容生成"""
    request = {"type": "bazi", "birth": "1990-06-15 10:00"}
    # 正官：接收请求
    routed = zhengguan_scheduler.route(request)
    # 元辰：定位模块
    module = yuanchen_locator.locate(routed)
    # 正财：检索知识
    knowledge = zhengcai_knowledge.search(module, routed)
    # 食神：生成内容
    result = shishen_generator.generate(knowledge, routed)
    assert result["day_master"] is not None
```
