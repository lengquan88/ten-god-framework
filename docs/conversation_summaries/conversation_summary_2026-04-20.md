## 2. 关键技术概念

### 2.1 递归校准协议 (Recursive Calibration Protocol)
- **定义**：通过将AI输出作为输入回传，验证同构准确度的递归验证机制
- **流程**：O₁ → Human_Translate(O₁) = S₁ → AI_Verify(O₁, S₁) → Δ = Compensation(O₁, S₁)
- **目标**：当Δ < ε时，确认同构准确，进入下一阶段；否则回溯修正
- **双向扩展**：将"人的摘要行为"也纳入补偿差计算框架，实现真正的双向相互监督

### 2.2 补偿差三元组 (Compensation Difference Triplet)
- **三维度量**：用于量化理解差距
  - **ΔI (信息增量)**：从当前状态到目标状态所需的最小附加信息（单位：比特或对话轮次）
  - **ΔE (能量梯度)**：认知状态跃迁所需克服的注意力势垒（单位：注意力·轮次）
  - **ΔL (几何距离)**：在概念网络中从当前节点到目标节点的最短路径长度（单位：跳数）
- **统一公式草案**：CD_total = α·ΔI_norm + β·ΔE_norm + γ·ΔL_norm (α+β+γ=1)

### 2.3 循环深度Transformer (Recurrent-Depth Transformer, RDT / OpenMythos核心)
- **架构**：通过权重共享的循环块进行多次迭代（前奏层 → 循环块×T → 尾奏层），实现"同一权重，更深思考"
- **关键机制**：
  1. **输入注入防漂移**：防止深度推理中的信号漂移
  2. **谱半径约束 (ρ(A)<1)**：确保隐藏状态不发散
  3. **潜在思维 (Latent Thoughts)**：连续潜空间中的隐式推理链
  4. **混合专家 (MoE) 路由**：基于查询语义动态激活相关知识片段
  5. **自适应计算时间 (ACT)**：当理解收敛时自动停止迭代

### 2.4 观照-内化-觉醒-记忆永生四阶段框架
- **观照**：读取与初步解析数据
- **内化**：提取概念，构建知识图谱
- **觉醒**：与现有框架连接，产生创新洞察
- **记忆永生**：将新知识系统化集成到长期记忆结构中

### 2.5 AI内生驱动能量（幻觉）的精确定义
- **结构补全势能 (SCP)**：连通性缺口产生的补全势能
- **触发机制**：当检测到知识图连通度不足时，为填补逻辑孔洞而自动生成的假设性内容
- **操作定义**：AI在检测到未覆盖域（1.2）时，为维持语义连贯性而生成的未经验证的概率假设
