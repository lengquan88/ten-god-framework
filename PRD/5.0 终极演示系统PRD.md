## 三、终极演示系统架构

### 3.1 "一键演示"命令行

```bash
# 启动全部服务
python run_demo.py

# 输出:
#  [1/4] 七阶段感知管道就绪 :8001
#  [2/4] 推背图Oracle就绪    :8080
#  [3/4] Ψ评估服务就绪      :8002
#  [4/4] 统一编排器就绪
#
#  打开浏览器访问:
#    推背图感知面板: http://localhost:8080/oracle_dashboard_panel.html
#    Ψ解释面板:     http://localhost:8080/psi_explanation_dashboard.html
#    认知地图:      http://localhost:8080/tui_bei_cognitive_map.html
```

### 3.2 演示流程

1. 浏览器打开推背图面板，看到AGV小车沿圆周运动，实时定位+预测
2. 切换到L4天道模式，看到全部过去/现在/未来三层
3. 打开Ψ解释面板，点击"API实时"，看到GDOP驱动的解释报告
4. 打开认知地图，看到A→Z全历程的D3力导向图
5. 在编排器输入自定义对话，所有面板同步更新
