## 一、为什么是MCP

MCP (Model Context Protocol) 是目前AI大模型连接外部工具的标准协议。将时空影像认知系统封装为MCP Server，意味着：

- Claude Desktop 可以直接调用 `/psi_evaluate` 评估对话质量
- DeepSeek 可以调用 `/oracle_query` 获取推背图预测
- 任何MCP客户端都是一等公民，无需适配

**一句话**: 51个模块从一个本地项目变成一个全球可用的AI工具。

## 三、MCP Server 实现

### 3.1 核心代码 (`mcp_server.py`, ~150行)

```python
#!/usr/bin/env python3
"""MCP Server: 时空影像认知系统"""
import json, sys, os
sys.path.insert(0, os.path.dirname(__file__))

class SpacetimeMCPServer:
    def __init__(self):
        from system_orchestrator import SystemOrchestrator
        self.orch = SystemOrchestrator()

    def handle_tool_call(self, name: str, args: dict) -> dict:
        if name == "psi_evaluate":
            result = self.orch.process_dialogue(args["question"], args.get("answer",""))
            return {"content": [{"type":"text","text":json.dumps(result.to_dict()["layers"]["cognition"], ensure_ascii=False)}]}
        elif name == "oracle_query":
            r = self.orch.process_dialogue("", "", target_position=(5,3,0))
            return {"content": [{"type":"text","text":json.dumps(r.to_dict()["layers"]["projection"], ensure_ascii=False)}]}
        elif name == "cognitive_style_profile":
            p = self.orch.profiler.get_profile(args["user_id"])
            return {"content": [{"type":"text","text":json.dumps(p.to_dict() if p else {},ensure_ascii=False)}]}
        # ...

    def run_stdio(self):
        """MCP stdio传输: JSON-RPC over stdin/stdout"""
        for line in sys.stdin:
            req = json.loads(line)
            if req.get("method") == "tools/list":
                resp = {"jsonrpc":"2.0","id":req["id"],"result":{"tools": TOOLS_SCHEMA}}
            elif req.get("method") == "tools/call":
                tool_name = req["params"]["name"]
                tool_args = req["params"].get("arguments", {})
                result = self.handle_tool_call(tool_name, tool_args)
                resp = {"jsonrpc":"2.0","id":req["id"],"result": result}
            sys.stdout.write(json.dumps(resp, ensure_ascii=False) + "\n")
            sys.stdout.flush()

TOOLS_SCHEMA = [
    {"name":"psi_evaluate","description":"评估对话认知意识水平...","inputSchema":{...}},
    {"name":"oracle_query","description":"推背图结构查询时空状态...","inputSchema":{...}},
    {"name":"cognitive_style_profile","description":"获取用户认知风格画像...","inputSchema":{...}},
]

if __name__ == "__main__":
    SpacetimeMCPServer().run_stdio()
```
