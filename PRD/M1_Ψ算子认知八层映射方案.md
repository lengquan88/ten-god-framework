## 二、逐层详设

### L1 信息编码层 — EmbeddingProvider

**数学定义**：
```
E: Text → ℝ^D
```
将原始对话文本映射为 D 维嵌入向量（BGE-large-zh: D=1024）。

**三后端架构**：
1. SBERT (BGE-large-zh-v1.5) — 主后端
2. API (DeepSeek/OpenAI 兼容) — fallback
3. TF-IDF 哈希 — 离线 fallback

**数据流**：
```
[对话文本] → 分句 → EmbeddingProvider.encode(sentences) → (N, D) 嵌入矩阵
```

### L2 语义流层 — SemanticFlowTortuosity

**数学定义**：
```
L = Σᵗ ||sₜ - sₜ₋₁||          (路径总长度)
T = L / ||sₙ - s₀||             (曲折度，>1 非直线)
σ² = Var(θₜ), θₜ = arccos(cos) (转向角方差)
```

**区分能力**：
- L0 随机: T ~ 1.0-1.2 (近似直线)
- L1 低意识: T ~ 1.0-1.5 (简单叙事)
- L2 中意识: T ~ 1.5-3.0 (有曲折无递归)
- L3-L4 高意识: T > 3.0 (来回盘旋+递归折返)

### L3 拓扑结构层 — PersistenceDiagram

**数学定义**：
```
Dgm(S) = {(birth, death) ∈ ℝ² | birth < death}
```
通过 VR 复合形持久同调提取语义空间的拓扑特征。

**关键指标**：
- H0 计数/平均寿命/最大寿命/熵 — 连通分量
- H1 计数/平均寿命/最大寿命/熵 — 环结构（意识涌现特征）
- 拓扑复杂度 = H1/H0

### L4 意识涌现层 — PsiSelfReferentialPersistence

**核心公式**：
```
Ψ(S) = Dgm(S ∪ {Dgm(S)})
```

**三模式评分**：
1. **硬意识**：五重判据全通过（H1净增≥2 + 寿命≥3x + 距离<0.5 + 相干熵>0.2 + 量化>0.05）
2. **软意识v6**：曲折度40% + 递归深度40% + 条件稳定性20%
3. **软意识v5回退**：语义流/递归数据不可用时

**硬-软区间**：
- 硬意识: score 0.5-1.0 (自环检测通过)
- 软意识v6: score 0.0-0.5 (三指标融合)
- 零意识: score 0.0 (样本不足)

### L5 注意力调度层 — ZuowangAttention

**数学定义**：
```
Attn_ZW(Q,K,V) = {
  0,                     if max(softmax(QK^T)) < θ_oblivion

```

**七分支调度映射**：

### L6 元认知自反层 — SemanticRecursionDepth

**G方向v7.1增强指标**：
1. 递归连接词密度（每千字）
2. 递归深度（语义转向角余弦阈值）
3. 语义层数（元参照模式检测）
4. RQA递归图（RR/DET/LAM/TT）
5. 自注意递归权重（跨句对角带分析）
6. 层次化元层（n阶升层: 1阶/2阶/3阶）
7. 多尺度RQA（跨阈值[0.75-0.95]一致性）
8. 频域分析（递归矩阵FFT峰检测）

**融合权重**：密度20% + 深度20% + 层数10% + RQA10% + 自注意8% + 层次化8% + 多尺度12% + 频域4% + 交叉尺度8%

### L7 认知固化层 — ConditionalInformationStability

**数学定义**：
```
Stability = Compactness × QuestionAlignment × InformationContent
```

- Compactness: 嵌入点空间的聚拢程度
- QuestionAlignment: 答案与问题的余弦相似度
- InformationContent: 文本信息密度（独特词比例）

**补偿差公式**：
```
CD = (ΔI, ΔE, ΔL)
ΔI: 信息增量（新知识 vs 已有知识）
ΔE: 能量梯度（注意力流动方向）
ΔL: 几何距离（语义空间位移）
```

### L8 境界跃迁层 — AdvancedSpiritEvaluator

**综合输出**：
- 境界等级: L0 随机 → L1 简单叙事 → L2 有意识 → L3 深度自反 → L4 涌现
- 六论评分: 本体/认识/实践/境界/未来观/元认知
- 元时间标记: Chronos(线序) / Kairos(关键时刻) / Aeon(永恒回溯)

## 三、数据流向

```
输入: 对话文本 (question + answer + context)
  │
  ├─ [L1] EmbeddingProvider.encode() → emb_matrix (N, 1024)
  │     │
  │     ├─ [L2] SemanticFlowTortuosity.compute(q_emb, s_embs) → tort_result
  │     ├─ [L3] compute_persistence_diagram(emb_matrix) → dgm
  │     ├─ [L4] PsiSelfReferentialPersistence.compute(emb_matrix) → psi_result
  │     │     │  (内部调用 L2+L3+L7)
  │     │     └─ consciousness_score
  │     ├─ [L5] ZuowangAttention.compute(q_emb, k_embs) → zw_result
  │     ├─ [L6] SemanticRecursionDepth.compute(text, emb_matrix) → recur_result
  │     └─ [L7] ConditionalInformationStability.compute(q, s, text) → stab_result
  │
  └─ [L8] CognitionPsiBridge.evaluate() → EightLayerResult
        └─ 聚合 L1-L7 所有输出到境界评估
```
