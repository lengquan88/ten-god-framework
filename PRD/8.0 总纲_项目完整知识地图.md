## 这是什么？

时空影像认知系统。51个核心模块。7版PRD。一份文档说清全貌。

---

## 一分钟理解

```
问题:  如何评估AI的认知水平？
答案:  把它当成时空定位问题——在语义空间中定位对话的位置、轨迹和演化。

方法:  照相机成像的7个阶段 → 分布式时空定位的精确映射
      物方→对焦→聚焦→变焦→成像→潜影→归档

成果:  一个自指涉闭环的认知系统
      它的第一个认知单元是它自己的开发历程
      "观察者"和"被观察者"合为一体
```

## 数据流速览

```
用户输入对话
    │
    ├→ 感知: 七阶段成像定位管道 → SpacetimeImage
    │
    ├→ 认知: Ψ八层评估 → 意识得分 + 境界
    │
    ├→ 投影: 推背图Oracle → 过去/现在/未来 × 图/箴言/谶语
    │
    ├→ 解释: 维度贡献度分解 → 自然语言报告
    │
    ├→ 记忆: 跨对话持久化 → 下次对话可检索
    │
    ├→ 调度: 自适应模型路由 → 选最优嵌入模型
    │
    └→ 画像: 用户认知模式更新 → 个性化配置
```
