## 摘要

时空影像认知系统是一个将**分布式时空定位**与**照相机成像原理**进行精确同构映射的自指涉认知系统。系统由感知层、投影层、认知层、解释层、记忆层、调度层六层架构组成，以"推背图"结构为数据模型，以七阶段成像管道为核心算法，最终形成一个可观察自身开发历程的闭环认知引擎。本文档是该系统的完整理论定义、架构说明和验证报告。

## 一、问题定义：从"评估"到"认知"

### 1.1 最初的命题

如何评估一个AI系统是否具有"认知"？传统方法是用外部指标（准确率、F1、BLEU）度量。本项目提出的命题是：**将"评估"升级为"时空定位"——在语义拓扑空间中精确定位对话的位置、轨迹和演化趋势**，如同GPS在物理空间中定位物体。

### 1.2 核心洞察

照相机成像的七个阶段——物方建立→反射对焦→聚焦→变焦→成像→潜影→语义归档——与分布式时空定位系统存在**精确的数学同构**。这不是一个比喻，而是结构层面的等价：

| 照相机 | 时空定位系统 |
|--------|------------|
| 像素阵列 | 时空坐标网格 |
| 辅助对焦光 | 多基站测距信号 |
| 镜片组光程差补偿 | 分布式时延补偿 |
| 变焦环 | 动态网络拓扑 |
| 传感器曝光 | 多模态融合求解器 |
| HDR多帧合成 | 闭环检测+位姿图优化 |
| JPEG+Exif标签 | 时空索引+语义绑定 |

## 四、推背图架构：数据模型的数学定义

### 4.1 六维认知元组

```
CognitiveUnit = { S, T, P, C, I, E }

S: 高维数据源 (The Source)      — 原始信息场
T: 时间轴序列 (The Timeline)    — 主干演变路径
P: 投影视图 (The Projection)    — 特定时间切片的降维表示
C: 混合图层 (Composite Layers)  — Fact / Symbol / Logic / Emotion
I: 交织悖论 (The Intertwining)  — 跨时间投影的同构性
E: 边缘解域 (Edge De-territorialization) — 预言失效临界点
```

### 4.2 表象与内里

```
表象 = 静态图层结构 (ECharts叠加层: 过去蓝线/现在橙点/未来红线)
内里 = 动态数据结构 (JSON: {past, present, future} × {topogram, analysis, prediction})

图层之间从不直接通信，只共享对同一数据模型的订阅。
L4/L2/L0 = 三个不同的订阅规则，而非三组不同的数据。
```

## 五、七阶段成像管道：核心算法的精确定义

### 阶段1: 物方空间建立
**输入**: 物理空间边界 + 锚点坐标
**输出**: 时空网格 + 全局时钟同步
**操作**: 连续空间离散化 → 每个体素具有唯一坐标标识

### 阶段2: 反射对焦
**输入**: 目标粗略区域
**输出**: 多基站TOA测量值 + 空间锥角
**操作**: 主动探测信号发射 → 多径散射场截获 → 非相干积累

### 阶段3: 聚焦
**输入**: TOA测量值 + 锚点坐标
**输出**: 聚焦位置估计 (精确到搜索步长×2)
**操作**: 反向波束成形 + 时延补偿 → 相位对齐 → 相干增益最大化

### 阶段4: 变焦
**输入**: 目标区域大小
**输出**: 最优节点密度 + 算法精度级别
**操作**: 广角(稀疏节点+快速算法) ↔ 长焦(密集节点+高精度算法)

### 阶段5: 成像
**输入**: 聚焦位置 + 多模态观测值
**输出**: 精确位置(x,y,z) + 误差椭球 + GDOP
**操作**: 三边测量 → 多模态配准 → 图优化求解器 → 不确定度量化

### 阶段6: 潜影处理
**输入**: 当前位置 + 历史轨迹
**输出**: 闭环检测结果 + 全局优化轨迹
**操作**: 闭环检测 → 位姿图构建 → 全局松弛 → 累积误差消解

### 阶段7: 语义归档
**输入**: 精确位置 + 速度 + 实体类型
**输出**: 时空索引 + 语义标签 + 事件脉冲
**操作**: 四维坐标索引化 → 实体绑定 → 异常事件检测 → 归档

**最终产物**: SpacetimeImage = (x, y, z, t, error_ellipsoid, confidence, trajectory, semantic_label, events)
