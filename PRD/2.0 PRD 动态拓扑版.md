1. **追踪**：持久特征的诞生与消亡（何时、何地出现认知洞见）
2. **刻画**：拓扑结构的定量描述（认知空间的形状如何变化）
3. **比较**：不同时间段的拓扑差异（从形似到神似的跃迁路径）

## 二、五大核心理论支柱

### 2.1 持久同调（Persistent Homology）

**原理**：通过 Vietoris-Rips 过滤构建单纯复形的嵌套序列，追踪拓扑特征的 birth-death 周期。

```
过滤参数 ε: 0 → ε₁ → ε₂ → ... → εₙ → ∞
单纯复形:   K₀ ⊆ K₁ ⊆ K₂ ⊆ ... ⊆ Kₙ = K∞
Hₖ 特征:   (birth_ε, death_ε) 持续对
```

**核心性质**：
- **H₀**：连通分量（语义簇的诞生与合并）
- **H₁**：环/空洞（语义循环结构、自指涉结构）
- **瓶颈距离稳定性**：输入扰动对持久图的影响可被瓶颈距离上界控制

**实现**：`topo_semantic/core.py` — `compute_persistence_diagram()` 双引擎自动切换
- ripser：精确计算（Vietoris-Rips 过滤）
- scipy：层次聚类近似（single/complete/average linkage + 合并距离差异）

### 2.2 锯齿持久同调（Zigzag Persistence）

**原理**：允许过滤方向在递增和递减之间任意切换，适用于数据点增删场景（对话节点的添加与剪枝）。

```
方向序列:  ↗ → ↘ → ↗ → ↘ → ...
复形序列:  K₁ → K₂ ← K₃ → K₄ ← K₅ ...
```

**关键文献**：Carlsson & de Silva 2010，将持久同调从单调过滤推广到任意方向。

**实现**：`time_topology_evolution.py` — `ZigzagPersistence` 类
- 边界矩阵约化（简化版，模拟方向切换对持久特征的影响）
- 适用场景：对话树节点的动态增减、语义空间的非单调演化

### 2.3 动态度量空间（Kim-Memoli 框架）

**原理**：将时间维度显式纳入拓扑分析，提出三参数时空过滤。

**三参数过滤**：F(X, t, α, τ) = Rips(X_t, α) × I_t，其中：
- X_t 是时间 t 的度量空间快照
- α 是尺度参数（控制连通半径）
- τ 是时间窗口参数（控制时间平滑性）

**距离度量**：λ-松弛交错距离 d_dyn(X, Y) = min{λ | 存在λ-交错}

**关键文献**：Kim & Memoli 2018，建立了动态度量空间的理论框架。

### 2.4 准锯齿持久同调 QZPH（Dey & Samaga 2026）

**原理**：在偏序集 ZZ×Z 上定义准锯齿过滤，将时间演化和尺度演化统一处理。

```
偏序集:  ZZ × Z = {((t, i), ε) | ε₁ → ε₂ → ... 或 ε₁ ← ε₂ ← ...}
                         ↑             ↑
                   正向时间演化    逆向时间演化
```

**关键文献**：Dey & Samaga 2026，最新理论突破。

### 2.5 动态图论（Dynamic Graph Theory）

**原理**：将对话结构建模为随时间演化的图，追踪图拓扑特征的演化。

**DTDG（Dynamic Time-Dependent Graph）**：
- 快照序列：G₁, G₂, ..., G_T（每个时间步的语义相似图）
- 时序图：G = (V, E, τ)，边带时间戳标签 τ: E → ℕ

**动态图拓扑特征（12维向量）**：
| 维度 | 名称 | 数学定义 |
|------|------|---------|
| 1 | 节点数 | n(t) |
| 2 | 边数 | m(t) |
| 3 | 平均度 | 2m(t)/n(t) |
| 4 | 连通分量数 | c(t) |
| 5 | 最大连通分量大小 | max_cc(t) |
| 6 | 聚类系数 | C(t) = 3×△(t) / τ(t) |
| 7 | 平均最短路径 | L(t) |
| 8 | 图直径 | D(t) |
| 9 | 度熵 | H_deg(t) = -∑p(k)log p(k) |
| 10 | 模块度 | Q(t) |
| 11 | 核数 | k_core(t) |
| 12 | 图谱半径 | ρ(L(t)) |

## 三、五大算法实现

### 3.1 滑动窗口嵌入（Sliding Window Embedding）

**功能**：将一维时间序列转化为高维点云（Takens 嵌入），用于持久同调分析。

**数学定义**：
```
SW_{M,τ}(x)(t) = [x(t), x(t+τ), x(t+2τ), ..., x(t+Mτ)] ∈ R^{M+1}
```

**参数**：
| 参数 | 默认值 | 说明 |
|------|--------|------|
| dimension M | 3 | 嵌入维度 |
| delay τ | 1 | 延迟步长 |
| stride | 1 | 滑动步长 |

**最优延迟选择**：
- 互信息第一极小值（推荐）
- 自相关第一过零点（回退）
- 卡普兰检验（嵌入维度和延迟联合优化）

**在评估系统中的应用**：
- 将语义流曲折度的时间序列转换为点云
- 检测对话语义空间中的周期结构和异常模式

### 3.2 时变持久同调（Time-Varying Persistence）

**功能**：追踪持久特征随时间的演化，检测拓扑结构的持续性和突变点。

**Vineyards 追踪**：Cohen-Steiner et al. 2006 — 持久特征在参数空间中的轨迹形成"葡萄藤"结构。

**持久景观（Persistence Landscape）**：Bubenik 2015 — 将持久图转换为可进行统计分析的函数空间表示。

**算法流程**：
```
输入: 嵌入点云序列 {X_t}_{t=1..T}
  ↓
为每个时间步计算持久图 PD_t
  ↓
追踪 H₀ 和 H₁ 特征的 birth/death 轨迹
  ↓
计算持久景观 λ_k(t, ε)
  ↓
输出: 特征演化轨迹 + 拓扑稳定性指标
```

### 3.3 CROCKER Plot

**功能**：在时间-尺度平面上可视化 Betti 数的演化，揭示拓扑结构的时变模式。

**CROCKER = Contour Realization Of Computed K-dimensional hole Evolution in Reduced coordinates**

**数学表示**：
```
CROCKER(t, ε) = β_k(t, ε)
```
其中 β_k(t, ε) 是时间 t、尺度 ε 下的第 k 个 Betti 数。

**在评估中的应用**：
- β₀(t, ε)：语义簇的合并与分裂模式
- β₁(t, ε)：语义循环结构的出现与消失
- 横截面(固定t)：当前认知状态的拓扑快照
- 纵截面(固定ε)：特定尺度下的拓扑演化轨迹

**演示结果**（2000采样点，94时间步）：
- H₁特征数范围：2~13
- CROCKER β₀范围：[0,79]，β₁范围：[0,8]

### 3.5 动态图拓扑演化

**12维特征向量**的时序分析，结合**时空图拉普拉斯**：

```
L_st = L_spatial + α·L_temporal
```

其中 α 控制时间平滑性，L_spatial 是当前时间步的图拉普拉斯，L_temporal 连接相邻时间步的节点。

**特征值谱分析**：

- 特征值变化率 → 拓扑突变检测
